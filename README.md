# 🟣 СІМІ Dashboard

## Як запустити на будь-якому ПК (Windows)

### Крок 1 — Встановіть Python (один раз)
1. Перейдіть на https://www.python.org/downloads/
2. Завантажте **Python 3.11** або новіший
3. Запустіть інсталятор
4. ⚠️ **ОБОВ'ЯЗКОВО** поставте галочку **"Add Python to PATH"**
5. Натисніть Install Now

### Крок 2 — Запустіть дашборд
Двічі клікніть на файл:
```
▶ Запустити СІМІ.bat
```

При першому запуску автоматично встановляться всі необхідні бібліотеки (~1-2 хвилини).
Браузер відкриється автоматично.

---

## Склад папки

| Файл | Призначення |
|------|-------------|
| `▶ Запустити СІМІ.bat` | Головний файл запуску |
| `launcher.py` | Запускає сервер і відкриває браузер |
| `app.py` | Код дашборду |
| `README.md` | Ця інструкція |

---

## Що потрібно для роботи

- Windows 7 / 8 / 10 / 11
- Python 3.11+ (безкоштовно, python.org)
- Інтернет при першому запуску (для завантаження бібліотек)
- Наступні запуски — без інтернету

## Для графіків в Excel-експорті

Графіки вбудовуються автоматично якщо встановлено `kaleido` (встановлюється автоматично через .bat).
